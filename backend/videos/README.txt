このフォルダにビデオファイルを入れてください。
Put your video files in this folder.

=== SUPPORTED FORMATS ===
.mp4, .mkv, .avi, .mov, .wmv, .flv, .webm, .m4v, .ts

=== THUMBNAILS ===
To add custom thumbnails, create a "thumbnails" subfolder:
  videos/
    thumbnails/
      MyVideo.jpg   (same name as video, .jpg or .png)
    MyVideo.mp4

The server will auto-generate SVG placeholder thumbnails if no
custom thumbnails are found.

=== EXAMPLE STRUCTURE ===
videos/
├── thumbnails/
│   ├── Movie1.jpg
│   └── Series_EP01.png
├── Movie1.mp4
├── Series_EP01.mkv
└── Documentary.avi
